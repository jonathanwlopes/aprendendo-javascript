Arrays (objetos disfarçados)

São objetos disfarçados, que possuem seus proprios metodos e propriedades

Uma das propriedades dos arrays é o length, que conta a quantidade de itens que tem no array

Um método do array é o push, que adicionar um item ao final do array: array.push('valor')

Para adicionar itens no meu array tenho algumas formas: array[indice] = 10

Um detalhe importante é array = ['','','',''] tenho um array de quatro posições, mas se eu utilizar o comando array[indice] = 10 será adicionada o valor ao 10º indice, deixando os anteriores vaziu ou seja array = ['','',...['10']

Lembando que ao chamar o array.push = ('valor'), adicionamos ao final da lista

Para remover o ultimo valor do array, podemos chamar a sintaxe array.pop()

Metodos interessantes de array

array.join()Este comando faz a junção dos itens do array, podendo passar como parametro um espaço exemplo: console.log('Meu almoço hoje será: ' array.join(' ')) vai me retornar a frase mais todos os itens do array

Temos também o array.reverse() onde os valores do array são invertidos, este método modifica o array e possui efeito colateral.

Temos o array.sort()este método vai ordenar os itens do array por ordem alfabetica e também possui efeito colateral

Outro método dos array é o array.toString()que me devolve o array em formato de string, separado por uma virgula, este método não tem efeito colateral, a diferença do join('') para o toString() é que no join consig definir qual é o separador, espaço, dois pontos, virgula etc..

Outro método é o array.concat([5,6,'Jonathan']) neste caso ele vai concatenar meu array atual, com os valores passados por parâmetro, não gerando efeito colateral. ()

Adicionar item no inicio do array, utilizamos a sintaxe array.unshift('valor'), onde será adicionado esse novo valor ao inicio do array, o oposto para adicionar no final usamos o array.push(''), para remover o ultimo array.pop()

Outro método é o array.shift() remove o primeiro item do array.

Método array.slice()este método possui dois parâmetros, o indice de inicio e de final, ele vai retornar todos valores entres este itens: array.slice(3,10), será retornado todo valor do indice 3 até o índice 9, pois ele não mostra o ultimo indice, este comando não tem efeito colateral.

Temos o método array.splice() que é muito diferente do slice(), o método splice ele tem efeito colateral, ou seja ele modifica o array principal, e ele pode tanto adicionar como remover itens do array

Método array.splice()

array.splice()O primeiro parametro define onde vai ocorrer a ação, de inclusão ou remoção daquele valor.

array.splice(3) --> neste caso o splice vai remover, tudo que tiver do indice três para frente.

array.splice(1,3) --> neste caso eu começo a remover do indice 1 e quantos valores eu quero remover aqui no exemplo 3 valores a partir do indice 1

Adicionar itens a partir de array.splice()

array.splice(1,0, 'a') neste exemplo, a partir do indice 1 não removemos nenhum item(0), e adicionamos um, novo valor ('a')

Aqui temos um for personalizado para arrays, com o método array.forEach ficando assim mais fácil a leitura do código

Temos também o método every, onde ele vai verificar se todos os itens do array são válidos

Temos também o método some, onde ele vai verificar se um dos itens é verdadeiro.

Mais um método é o array.map que percorre todo o array também como os método anteriores, porem ele cria um novo array

Temos também o método array.filter() que irá filtrar os itens do array, retornando um novo array filtrado

Mais um método interessante é o array.indexOf(4,1) onde a partir do índice 1 o método vai pesquisar pelo valor 4 no array

Uma forma interesante de utilizar o indexOf é: array.indeOf(4) > -1 desta forma ele vai retornar true ou false ou seja existe ou não o valor no array.

Object

Os objetos possuem uma maneira de Herança, onde utilizando a sintaxe. obj2.Object.create(obj) O objeto 2 vai receber todas as propriedades do obj

O objeto também posssui um método chamado hasOwnProperty, que vai verificar se o objeto tem a propriedades (desta forma, esse metodo é muito útil para verificar apenas as propriedades do objeto, sem as heranças), obj2.hasOwnPropety('x')

Outro método dos objetos é o Object.keys(obj), este método retorna o seu objeto em formato de array, assim possibilitando utilizar as propriedades de objeto em array.

Podemos confertir se um objeto é prototipo de outro utilizando a sintaxe obj.isPrototypeOf(obj2)

Outro método é o let str = JSON.strigify(obj) que vai transformar o obj em string, para realizar o processo inverso transformar string em obj utilizamos JSON.parse(str)
