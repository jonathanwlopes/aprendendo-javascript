React, Angular, jQuery, são linguagens baseadas em Javascript, sua utilidade são bibliotecas para facilitar a programação, mais como todas utlizam javascript em sua base temos um grande motivo e primordial entender e dominar a linguagem pai javascript, para somente assim utilizar essas ferramentas.

Entendendo Variaveis

Imaginar um terreno, e dentro deste terreno temos de organizar as vagas pois se não organizar não teremos controle.

Para organizar criamos a variavel vaga e lemos que esta vaga recebe um carro exemplo var vaga1 = 'carro1' (leia-se vaga recebe carro) nunca diga que a vaga é igual a carro.

No JS, temos diversos tipos de dados, como number, string, boolean, null, undefined, object (vetores), function.

Uma informação importante é: a varíavel pode mudar o seu tipo typeof, durante a execução do programa, basta definir, exemplo: var numero = 16 e na sequencia dizer que var numero = '16' seu tipo typeof deixou de ser number para ser uma string.

Tratamento/Manipulação de dados

Para executar a conversão de dados de string para inteiro ou float, devemos utilizar a seguinte formula: var n1 = Number.parseInt(window.prompt('Digite o número')) Esta é uma forma antiga de utilizar hoje conseguimos utilizar somente Number para realizar a conversão.

Nas versões atuais de javascript, podemos utilizar para concatenar as informações a técnica de template strings onde vamos realizar desta forma: `Eu estou aprendendo ${nomeVariavel}`

Alguns comandos para manipular os dados são o var.length que me retorna a quantidade de caracter da variavel(em números) temos também o variavel.toUpperCase()que vai me retornar a string em letras maiusculas temos também o variavel.toLowerCaseCase()que vai me retornar a string em letras minusculas

Temos algumas formas de formatar os dados com javascript. um comando útil para imprimir na tela é: document.write(`O que desaja imprimir ${variavel}`)

Conseguimos também formatar numeros o comando que utilizamos é: variavel.toFixed(2).replace('.',',') neste caso teremos o resultado onde será incliudo duas casas apos a virgula, e trocado o . por ,

Outro comando útil para formatação de número é o variavel.toLocaleString('pt-br', {style: 'currency', currency: 'BRL'})
